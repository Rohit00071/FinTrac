Create a complete finance management app called "FinTrack" for my MCA final year project using React Native Expo. Here's everything the app needs:

## *🎯 APP OVERVIEW*
A personal finance tracker with expense monitoring, budgeting, savings goals, and financial analytics.

## *📱 MUST-HAVE FEATURES*

### *1. USER AUTHENTICATION*
- Login/Register screens
- Email/password authentication
- Profile management
- Session persistence

### *2. DASHBOARD*
- Total balance display
- Income vs Expense pie chart
- Monthly spending trend graph
- Recent transactions list
- Quick stats (Weekly/Monthly totals)
- Upcoming bill reminders
- Savings goals progress

### *3. TRANSACTION MANAGEMENT*
- Add income/expense with:
  * Amount, category, date, notes
  * Receipt photo capture
  * Recurring option
- Categories: Food, Transport, Shopping, Bills, Entertainment, Healthcare, Education, Others
- Edit/delete transactions
- Search and filter by date/category
- Transaction history view

### *4. BUDGET SYSTEM*
- Set monthly budget per category
- Visual progress bars (green/yellow/red)
- Alerts when reaching limits (80%, 90%, 100%)
- Budget vs actual comparison
- Month-to-month tracking

### *5. SAVINGS GOALS*
- Create goals (car, vacation, emergency fund)
- Set target amount and deadline
- Track progress with circular chart
- Make contributions
- Goal completion celebration

### *6. BILLS & REMINDERS*
- Add recurring bills (rent, utilities, subscriptions)
- Due date notifications
- Mark as paid
- Payment history

### *7. REPORTS & ANALYTICS*
- Monthly/yearly reports
- Category-wise spending analysis
- Income vs expense trends
- Export reports (PDF/CSV)
- Spending insights

### *8. MULTI-ACCOUNT SUPPORT*
- Cash, Bank Account, Credit Card, Digital Wallet
- Transfer between accounts
- Account balance tracking

### *9. SETTINGS*
- User profile (name, email, currency)
- Theme (Light/Dark mode)
- App lock (PIN/Password)
- Data backup & restore
- Notification preferences
- About & help

### *10. ADDITIONAL FEATURES*
- Receipt scanning (camera/gallery)
- Financial tips/insights
- Expense sharing (split bills)
- Investment tracking (mock)
- Multi-language support (optional)

## *🎨 UI/UX REQUIREMENTS*
- Clean, modern interface
- Consistent color scheme:
  * Primary: Green (#4CAF50) for income
  * Danger: Red (#F44336) for expenses
- Charts and graphs for data visualization
- Loading animations and skeletons
- Empty state illustrations
- Swipe gestures for actions
- Responsive on all screen sizes

## *💾 DATA TO STORE*
- User profile
- All transactions with categories
- Budget limits and spending
- Savings goals
- Bills and reminders
- App settings

## *🔧 TECH STACK*
- React Native with Expo
- TypeScript for type safety
- React Navigation (Stack + Bottom Tabs)
- Redux Toolkit for state management
- AsyncStorage for local data
- React Native Paper for UI components
- React Native Chart Kit for graphs
